# 📊 Linear Regression from Scratch (Without Sklearn)

This project demonstrates the implementation of Simple Linear Regression from scratch using Python, without relying on any machine learning libraries.

---

## 🚀 Objective

To understand the core mathematics behind Linear Regression by manually implementing slope (m) and intercept (b).

---

## 📁 Dataset

- `placement.csv`
- Features:
  - CGPA (Independent Variable)
  - Package (Dependent Variable)

---

## ⚙️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Matplotlib  

---

## 🧠 Approach

The model is built using the mathematical formulas:

- Slope (m):
  m = Σ(x - x̄)(y - ȳ) / Σ(x - x̄)²  

- Intercept (b):
  b = ȳ - m x̄  

---

## 🔧 Steps Performed

1. Data loading and preprocessing  
2. Manual calculation of slope and intercept  
3. Model training using custom class  
4. Prediction on test data  
5. Visualization of regression line  

---

## 📈 Results

The custom-built model successfully captured the relationship between CGPA and Package.

---

## 📌 Key Learnings

- Deep understanding of Linear Regression  
- Mathematical intuition behind ML models  
- Debugging real-world coding errors  
- Difference between manual and library-based approaches
